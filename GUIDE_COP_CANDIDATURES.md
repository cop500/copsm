# Guide COP Space — Traiter les candidatures

**Application :** https://copsm.space  
**Public :** équipe COP (conseillers, conseillère carrière, managers)

---

## En bref

| Étape | Où dans l'app | Action |
|-------|---------------|--------|
| 1 | **Demandes entreprises** | Recevoir et traiter la demande de l'entreprise |
| 2 | **Stagiaires → Candidatures reçues** | Voir toutes les candidatures |
| 3 | **Stagiaires → Candidatures par demande** | Trier les CV et envoyer le lot |
| 4 | **Stagiaires** (les deux vues) | Suivre jusqu'à acceptation ou refus |

---

## 1. Connexion

1. Aller sur **https://copsm.space**
2. Se connecter avec vos identifiants
3. Cliquer sur **Candidatures** ou **Stagiaires** dans le menu

---

## 2. Demande entreprise

**Chemin :** Menu → **Demandes entreprises**

1. Ouvrir la nouvelle demande
2. **Assigner** un responsable
3. Changer le statut : *En attente* → *En cours* → *Terminée*
4. Ajouter un **commentaire interne** si besoin (visible uniquement par l'équipe)

**Lien à donner à l'entreprise :** https://copsm.space/demande-entreprise

---

## 3. Candidatures reçues

**Chemin :** **Stagiaires** → onglet **Candidatures reçues**

- Liste de **toutes** les candidatures
- **Rechercher** : nom, entreprise, poste
- **Filtrer** : statut, date, pôle, filière
- **Ouvrir le CV** (PDF dans l'application)
- **Changer le statut** et ajouter des notes

### Statuts à utiliser

| Statut | Signification |
|--------|---------------|
| Envoyée | Candidature reçue |
| En cours d'étude | Analyse en cours |
| Invitée en entretien | Convocation envoyée |
| Entretien programmé | Date fixée |
| Entretien réalisé | Entretien fait |
| Acceptée | Candidat retenu |
| Refusée | Candidat non retenu |

**Lien à donner aux candidats :** https://copsm.space/candidature

---

## 4. Tri des CV et envoi à l'entreprise

**Chemin :** **Stagiaires** → onglet **Candidatures par demande**

Les candidatures sont classées par **entreprise** puis par **poste**.

### Pour chaque candidat

1. Cliquer **CV accepté** ou **CV refusé**
2. Vérifier le badge : *À trier* / *CV accepté* / *CV refusé*
3. Le tri est **sauvegardé** — il reste après déconnexion

### Envoyer le lot à l'entreprise

1. Terminer le tri sur le dossier
2. Cliquer **Télécharger les CV acceptés (ZIP)**
3. Seuls les CV **acceptés** sont dans le fichier
4. La date **1er envoi** s'affiche sur chaque fiche
5. Les envois suivants sont numérotés (*2e envoi*, *3e envoi*…)

> **Important :** trier les CV **avant** de télécharger le ZIP.

---

## 5. Suivi jusqu'à la fin

Mettre à jour le **statut** de chaque candidat au fil du processus :

```
Envoyée → En cours d'étude → Entretien → Acceptée ou Refusée
```

Utiliser **Candidatures reçues** (vue globale) ou **Candidatures par demande** (vue par dossier).

---

## Aide rapide

| Question | Réponse |
|----------|---------|
| Où trier les CV ? | Stagiaires → **Candidatures par demande** |
| Où voir toutes les candidatures ? | Stagiaires → **Candidatures reçues** |
| Où traiter la demande entreprise ? | **Demandes entreprises** |
| Le tri disparaît après déconnexion ? | Non, il est enregistré en base |
| Problème technique ? | Contacter l'Admin COP |
